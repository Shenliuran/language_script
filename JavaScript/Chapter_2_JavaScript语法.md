# 第二章 JavaScript语法

## 准备工作

+ 编写前期准备代码：

    ```html
    <!DOCTYPE html>
    <html lang = "en">
    <head>
        <meta charset="utf-8">
        <title>Example</title>
    </head>
    <body>
        Mark-up goes here...
        <script src="file.js"></script>
    </body>
    </html>
    ```

    这是浏览器加载最快的方法

+ JavaScript是弱类型语言，这意味着程序员可以在任何阶段改变变量的数据类型
